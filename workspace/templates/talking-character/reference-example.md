# Reference example — placeholder

Этот файл заполняется после первого реального проекта на template
`talking-character`. Сейчас пуст.

## Что положить сюда после первого ship'a

- Project ID + краткое описание
- Persona archetype + конкретные параметры
- Setting + scene-specific notes
- Final hook text (что сработало, что нет)
- VO script (full text)
- Final captions choice (какой компонент использовали и почему)
- Music brief + final track
- Cost breakdown (фактический, а не оценка)
- Что бы делали по-другому в следующий раз

## Why we keep this empty for now

Темплейт — это **vibe-reference**, не fill-in-the-blank. Заполнять
reference-example.md без реального проекта = создавать fake данные, которые
введут будущие проекты в заблуждение.

Когда будет первый успешный talking-character проект, заменить этот файл
содержимым по структуре выше.
