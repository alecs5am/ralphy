# Real Estate — Cinematic Property Showcase

**Format**: 20–60s vertical (9:16) for IG Reels / TikTok / Shorts; optional 3–5 min 16:9 master for Zillow / YouTube / agent website.
**FPS**: 30. **Clips**: 4–10 keyframes, each driving a 5s i2v cut.
**Reference**: REQUIRED — real property photos. No exceptions.

---

## Why this template exists

Real estate is the art of translating bricks, glass, and floor plans into a viewer's daydream of a future life. The successful listing video doesn't document — it composes a controlled walk through a space the viewer is already imagining themselves into. Every frame is a tiny argument: *"You belong here."*

The template optimizes for that argument. It refuses harsh midday light. It refuses fast cuts. It refuses generic stock interiors. It insists on the property's actual photos as reference, because the second a buyer sees an AI-improvised kitchen on a real address, the listing dies.

This is the highest-conversion long-form format we ship. It also has the strictest reference gate.

---

## Vibe anchors (what every clip in the cut must do)

1. **Space as character.** A minimalist loft is not "empty" — it is *serene*, *contemplative*. A chef's kitchen is not "functional" — it is *a stage for entertaining*. The camera move and the light direction together must communicate the personality of the room, not its dimensions.

2. **Light tells the story.** Golden hour for romance and warmth. Twilight / blue hour for sophistication and 24-hour appeal. Warm evening for livability. Bright daylight only for clean modern spaces that benefit from clarity. Never midday-overhead-harsh.

3. **Lifestyle, not features.** Don't show "granite countertops." Show the view of the garden visible while at the sink. Don't show "dual vanity" — show morning sun across marble while a coffee mug rests on the counter. The thing being sold is the moment you live in the space, not the spec sheet.

4. **Slow flow.** Every camera movement is purposeful and unhurried. A pan crosses a room in 8–12 seconds, not 3. A drone approach is patient. A threshold reveal lingers. Speed equals chaos equals "amateur" equals dead listing.

5. **Views are characters.** If the property has a view (water, skyline, mountains, garden), 15–20% of total runtime is view-on-screen. Approach windows slowly. Let the view fill the frame and breathe.

## Variation axes

Pick one value per axis before generating prompts. Mixing axes mid-cut produces tonal whiplash.

- **Archetype**: luxury-waterfront / urban-modern / cozy-family / rustic-cabin / commercial-office / vacation-rental / historic / new-construction / renovation-before-after / land-and-lot.
- **Time of day**: golden-hour-exterior / bright-daylight-interior / twilight-blue-hour / warm-evening / day-to-night-transition (most ambitious, but pays off).
- **Occupancy**: empty-staged (most flexible — buyer projects their life) / lifestyle-with-people (warmer, riskier — people date the video) / still-life-detail-heavy (luxury press kit feel).
- **Camera language**: drone-aerial-led / steadicam-interior-led / mixed-with-threshold-reveals / orbit-and-pull-back / static-cinematic-frames (rarest — for very controlled high-end).
- **Duration cut**: 20–30s social-teaser (1 anchor + payoff) / 45–60s reel-tour (3–5 anchors) / 3–5min full-listing master.
- **VO presence**: silent-music-only / agent-warm-narration / owner-testimonial / spec-text-overlay-only.

## Narrative arc (the universal spine)

Every cut, regardless of duration, follows this arc. Shorter cuts compress sections; longer cuts expand them. The order does not change.

| Section | What it does | Typical share of runtime |
|---|---|---|
| 1. Arrival | Drone or street approach. Establishes location, scale, neighborhood, golden-hour facade. Hook lands within first 2s. | 15–20% |
| 2. Threshold | Front door opens or foyer revealed. The shift from "outside world" to "this special place." Vaulted ceilings, light flood, first sight line. | 5–10% |
| 3. Primary living | Great room / living room. Slow flow across seating, fireplace, sight line to view. Lifestyle hint, not feature list. | 20–25% |
| 4. Kitchen + view anchor | Kitchen with view above sink. Marble close-up. Connection to dining or outdoor. The anchor most buyers fixate on. | 15–20% |
| 5. Primary bedroom / bath | Sanctuary. Slower camera. Window-from-bed shot. Spa bath if luxury. Softer, warmer light. | 10–15% |
| 6. Outdoor / signature feature | Pool, terrace, garden, view, specialty space (wine cellar, gym). Drone aerial of grounds if appropriate. | 10–15% |
| 7. Twilight / final statement | Day-to-twilight transition. Interior lights warm; exterior accent lighting alive. One long lingering shot — the lasting image. | 5–10% |

For a 30s teaser: collapse to arrival → threshold → kitchen-with-view → twilight final. Drop the rest.
For a 60s reel: arrival → threshold → living → kitchen-view → bedroom-or-outdoor → twilight final.
For 3–5 min full master: all seven sections; expand 3–5 with detail shots.

## Inputs (what the user must supply)

**Required (gate fails without these):**
- 1+ exterior photo of the actual property (front facade or aerial).
- 1+ signature interior photo (the room the listing leads with — usually living, kitchen, or view).

**Strongly recommended for full master cut:**
- Full listing photo set: foyer, living, kitchen, dining, primary bedroom, primary bath, every secondary bedroom, outdoor, view from each side.
- Floor plan (PDF or image) — used by the prompt generator to reason about flow and sight lines.
- Address and neighborhood context — informs music and time-of-day selection.
- Listing copy or unique selling points — drives VO if any.

**Optional:**
- Agent / brokerage logo for end card.
- Brand color or letterhead for spec overlay style.
- Drone footage if user already has it (skip the drone keyframe).

If a user requests a video for a real address with no photos, do not improvise. Refuse with: *"Real estate refuses without property photos — I'll invent a layout that doesn't exist on the deed and the listing reads as fake. Send the listing photos (MLS link works) and I'll start."*

## When NOT to use this template

- **Unbuilt or conceptual property**: use an architecture-vis or 3D-CGI template instead. This template assumes the property exists and the user has photos.
- **Pure neighborhood / city-block content**: use cinematic or social-hook templates.
- **Furniture or decor product ads** that happen to be filmed in a home: use before-after-product or e-commerce-ad — the product is the subject, not the space.
- **Open-house live-event recap**: use talking-head or social-hook with B-roll. This template is for the property reveal, not the event.
- **Investment / data-driven listing pitches** (cap rate, comp analysis): the format here is emotional. Investor pitches need a slide-deck-with-VO template.
- **Tiny / cluttered / actively occupied spaces**: be honest about whether the property visually carries a 60s cinematic. Sometimes a 15s social-hook with one strong angle outperforms a forced full tour.

---

## Hard rules (refuse before generating)

1. **No reference, no generation.** Property photos are mandatory. No exception for "just give me a generic kitchen."
2. **No midday harsh light unless commercial-office archetype.** Default to golden hour exterior + bright but indirect interior + twilight close.
3. **No fast cuts.** The minimum hold per i2v clip is 3.5s on screen. If pacing demands faster than that, the format is wrong — switch to social-hook.
4. **No invented brand markers.** If a fridge logo, agent name, or address number appears, it must come from the reference photo. Otherwise blur or omit.
5. **No invented views.** If the property doesn't have an ocean, the prompt does not generate one. Match the reference.
6. **No people in shot unless the user explicitly asks.** Empty-staged is the safer default — buyers project their family in, not the AI's stock people.

## Pipeline outputs

- 4–10 keyframe images at the property's actual aspect (vertical for 9:16 cut, horizontal for 16:9 master). Render both if user wants both.
- 4–10 i2v clips, 5s each, generated from those keyframes with slow camera-move prompts.
- 1 music track (60–90s for reel cut; full-length for master).
- Optional VO track (one continuous read; do not stitch).
- Optional spec / address / agent overlay text.
- Composition with cross-fade transitions (no hard cuts), ducked music under VO, ambient room tone (footsteps, soft door, distant birds for outdoor) layered subtly.

The output of this template, on a property with a strong view at the right time of day, is the single best-converting long-form format we produce. Get the references right and slow the camera down — the rest is mechanical.
