# Hooks — fit-check

12 opening beats for fit-check / OOTD videos. Each is a 0-1s hook: the line that appears as on-screen text, the transition cue that fires after it, and the music vibe the cut-cadence assumes.

Don't paste a hook verbatim — they're rotation. If you've used #3 in the last week, pick something else. Audience-handles this format daily; they spot reuse.

## 1. "Today's fit"
- **Niche fit:** anyone, daily. The neutral default — works for streetwear, workwear, modesty.
- **Transition cue:** mirror-flash on the first downbeat after 1.0s.
- **Music vibe:** medium-energy (90-110 BPM). Hyperpop or amapiano.
- **Why it works:** zero ambition. The outfit speaks; the text is just a label.

## 2. "Three days, three fits"
- **Niche fit:** week-recap, capsule wardrobe, content series.
- **Transition cue:** snap-cut between each day, hard on the downbeat.
- **Music vibe:** drill or latin-trap. ≥ 110 BPM so all three cuts fit in 15s.
- **Why it works:** sets the count up front — viewer knows there are 3 payoffs and waits.

## 3. "Workwear edition"
- **Niche fit:** office, business-casual, "what I wore to work this week".
- **Transition cue:** jump-cut (3 micro-cuts in 0.4s) between fits, signaling pace.
- **Music vibe:** 90s-house or quiet-luxury bed (low BPM, sophisticated). 80-95 BPM.
- **Why it works:** niche-tagged in the title — the viewer self-selects.

## 4. "[N] under [price]"
- **Niche fit:** thrift, fast-fashion-haul, second-hand. Number + price = clickbait grammar.
- **Transition cue:** snap-cut, with a brand-tag large-overlay showing the price for each piece.
- **Music vibe:** trending-audio-of-the-week (TikTok haul-audio). 100-130 BPM.
- **Why it works:** value-promise in the hook. "5 fits under $50" — the audience opens to verify.

## 5. "Mirror selfie + fit"
- **Niche fit:** anyone, single-outfit hero. Most authentic-feeling.
- **Transition cue:** none — this is a 7-10s single-outfit video. One mid-clip mirror-flash for energy.
- **Music vibe:** chill r&b or 90s-house. 85-100 BPM.
- **Why it works:** declares mirror-pose framing in the title — sets the audience's eye.

## 6. "Get ready to be jealous"
- **Niche fit:** quiet-luxury, hero piece, statement-coat moment.
- **Transition cue:** mirror-flash on downbeat 1, slow spin on downbeat 2, hold.
- **Music vibe:** big-room hyperpop or filtered house. ≥ 120 BPM.
- **Why it works:** classic "watch this" provocation. Best on a real hero piece — flops on basics.

## 7. "Thrift haul: [N] pieces"
- **Niche fit:** second-hand, thrift, vintage. The "I just got back from Goodwill" energy.
- **Transition cue:** snap-cut per piece, with a price-tag corner overlay (e.g. `$8`, `$12`).
- **Music vibe:** indie-pop or amapiano. 95-115 BPM.
- **Why it works:** count + niche + value, all in 4 words.

## 8. "Modest fits ✿"
- **Niche fit:** modesty fashion. Flower / leaf glyphs in text are a niche-handle the audience recognizes.
- **Transition cue:** mirror-flash (gentle, 2-frame) — never jump-cut, which reads aggressive.
- **Music vibe:** dream-pop or instrumental indie. 80-100 BPM. No lyrics.
- **Why it works:** community-coded. The audience inside the niche immediately knows.

## 9. "POV: it's Monday"
- **Niche fit:** workwear, daily, "girl-going-to-the-office" content.
- **Transition cue:** jump-cut sequence as if the morning is rushing.
- **Music vibe:** drill or amapiano with a fast hi-hat. 110-130 BPM.
- **Why it works:** POV grammar = relatable framing; "Monday" = universal pain → audience leans in.

## 10. "Same dress, [N] ways"
- **Niche fit:** capsule, styling-showcase, sustainability angle.
- **Transition cue:** mirror-flash with the same garment, accessories rotating each cut.
- **Music vibe:** light electronic or 90s-house. 95-115 BPM.
- **Why it works:** the constraint is the hook. Viewer wants to see how the same piece transforms.

## 11. "Streetwear drop, this week"
- **Niche fit:** streetwear, sneaker-culture, "what I'm wearing this week".
- **Transition cue:** snap-cut, brand-tag large-overlay for each piece (sneakers especially).
- **Music vibe:** drill, trap, hyperpop. ≥ 110 BPM.
- **Why it works:** "drop" + "this week" = niche-vocabulary + recency. Streetwear audience clicks instantly.

## 12. "Outfit of the day · [day]"
- **Niche fit:** anyone, but especially modesty / workwear / daily-content series.
- **Transition cue:** none for a single-fit video, or one mirror-flash mid-clip.
- **Music vibe:** anything. The hook is so generic that the music carries the niche.
- **Why it works:** OOTD is the genre's literal name. Plain-text label, zero ambition, daily-content default.

## Picking a hook (cheat-sheet)

| If… | Use |
|---|---|
| It's a daily, single-outfit post | #1, #5, #12 |
| It's a thrift / haul / value-tagged post | #4, #7 |
| It's a workwear / office post | #3, #9 |
| It's a streetwear / hero-piece post | #6, #11 |
| It's a modesty post | #8, #12 |
| It's a styling / same-piece-many-ways post | #10 |
| It's a multi-day / week-recap post | #2 |
