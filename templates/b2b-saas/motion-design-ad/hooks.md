# Motion Design Ad — Hook Library

The first 2 seconds decide whether the viewer stays. Pick **one** hook (or chain two for a 3s opening) — never start with a fade-from-black, a logo card, or a slow zoom. Each hook below is a complete recipe: visual + motion verb + color cue + sound cue.

Hooks pair with visual styles (`Dark Premium`, `Clean Minimal`, `Abstract Data`, etc.). The hook supplies the motion; the style supplies the palette and texture.

---

## 1. Data-Visualization Explosion

A single metric (e.g., "500K active users") sits centered, then explodes outward — particles, secondary stats, glow trails.

- **Best for:** analytics, BI, fintech, dashboards, anything "numbers go up"
- **Motion verbs:** *explode outward*, *cascade down*, *radiate*, *shoot out in rapid succession*
- **Color cue:** dark base + electric accent (cyan, gold, magenta)
- **Sound:** whoosh on explosion + ascending chime per secondary metric (3-5 chimes, 0.1s apart)
- **Duration:** 1.2 - 1.8s
- **Watch out for:** too many metrics on screen at once. Cap at 5 secondary stats; more reads as noise.

## 2. UI Morph

An interface element transforms through 2-3 states (button → loading bar → checkmark; closed sidebar → open sidebar → settings panel).

- **Best for:** design tools, dev platforms, workflow apps, anything interaction-heavy
- **Motion verbs:** *smoothly morph*, *deform along bezier curve*, *transform between states*
- **Color cue:** brand primary on neutral background; element glows during the morph
- **Sound:** morph tone (smooth pitch transition matching morph duration) + soft click on final state
- **Duration:** 1.5 - 2s
- **Watch out for:** morphing into something abstract. The end state must be a recognizable product element, not a generic icon.

## 3. Code-to-Product Transformation

Terminal text / code lines compile, then explode into the actual product UI.

- **Best for:** developer tools, infra, APIs, AI/ML platforms
- **Motion verbs:** *compile*, *cascade upward*, *shatter into particles*, *re-gather as UI*
- **Color cue:** green-on-black terminal → vibrant brand-color UI; sharp contrast shift
- **Sound:** typewriter clicks (0.05s/char) + processing sound + electronic resolution tone on the morph
- **Duration:** 2 - 2.5s
- **Watch out for:** fake-looking code. Use real syntax (Python, JS, SQL, shell — match the audience). One-liner is fine; sloppy pseudocode is a defect.

## 4. Glitch-to-Clean

The screen flickers, distorts, shows error states for 0.6-0.8s, then **snaps** clean and pristine.

- **Best for:** problem-solution narratives, "fix the chaos" positioning, security, debugging tools, productivity-after-mess
- **Motion verbs:** *flicker*, *pixelate*, *distort*, *snap to clean state*
- **Color cue:** dark/red/broken → brand-clean palette on the snap
- **Sound:** glitch stutter (0.3s) + satisfying ding on resolution
- **Duration:** 1 - 1.5s
- **Watch out for:** overusing glitch elsewhere in the ad. Once per ad, max — otherwise the technique loses impact and starts to feel like a real bug.

## 5. Light-Beam Scan

A bright animated beam sweeps across a dark interface, illuminating features as it passes.

- **Best for:** premium SaaS, "see what's new", security, enterprise reveals
- **Motion verbs:** *sweep across*, *trace the edge*, *scan top-to-bottom*, *bloom on contact*
- **Color cue:** very dark background + single bright beam color (cyan, white, electric blue) with bloom
- **Sound:** laser sweep tone (ascending or descending pitch) matching beam duration
- **Duration:** 1.5 - 2s
- **Watch out for:** beam too thin or too fast. Eye needs ≥0.8s to register what's being illuminated.

## 6. Particle Burst from Logo

Brand logo appears, then explodes into particles that re-gather to form the product UI.

- **Best for:** product launches, brand-forward reveals, ecosystem stories
- **Motion verbs:** *shatter into particles*, *radiate outward*, *re-form as*, *coalesce into*
- **Color cue:** brand primary across particle field; UI palette on the re-form
- **Sound:** bass swell during gather + soft impact when UI lands
- **Duration:** 1.8 - 2.4s
- **Watch out for:** the logo must be on screen long enough to register before it shatters (≥0.4s static hold). Otherwise the brand association is lost.

## 7. Isometric Camera Zoom

3D isometric view of a system / ecosystem; camera dramatically pulls back (or pushes in) to reveal scale.

- **Best for:** B2B platforms, ecosystems, infra, "everything in one place" positioning
- **Motion verbs:** *pull back*, *zoom out*, *orbit slowly*, *reveal the full ecosystem*
- **Color cue:** Dark Premium with module-specific accent glows (one color per system component)
- **Sound:** rising electronic tone peaking at the reveal frame
- **Duration:** 2 - 2.5s
- **Watch out for:** too many modules. 4-6 max; more reads as a network diagram, not an ad.

## 8. Hand Interaction

A hand (animated or live-action) taps / swipes / gestures on a floating device; the device responds in real-time.

- **Best for:** mobile apps, touch interfaces, consumer SaaS
- **Motion verbs:** *tap decisively*, *swipe upward*, *pinch to zoom*, *gesture across*
- **Color cue:** Clean Minimal — white/light background, brand accent illuminates on contact
- **Sound:** light tap (0.1s) synced to contact + UI response sound
- **Duration:** 1.5 - 2s
- **Watch out for:** uncanny-hand. Either go fully stylized (illustration, simple silhouette) or use real footage as a reference — the in-between is creepy.

## 9. Metric Counter Rapid Climb

A single number rapidly counts from 0 (or low) to target (e.g., "Users: 0 → 50,000"), with audio sync.

- **Best for:** social proof, traction stories, "we grew this fast" positioning
- **Motion verbs:** *count up*, *climb rapidly with easing*, *snap to final value*
- **Color cue:** number in brand primary on neutral background; subtle glow on final value
- **Sound:** rapid ticker clicks during climb + impact tone on final value
- **Duration:** 1.2 - 1.8s
- **Watch out for:** number too small to read on mobile. Final value should be ≥40% of frame height.

## 10. Neon-Glow Activation

Dark interface suddenly comes alive — neon outlines trace UI elements, panels light up in sequence.

- **Best for:** gaming, crypto, music, edge-forward consumer apps
- **Motion verbs:** *illuminate*, *trace the outline*, *flicker on*, *bloom*
- **Color cue:** near-black + 2 neon accents (cyan + magenta, or pink + lime)
- **Sound:** electrical hum + sharp crackle as each element activates
- **Duration:** 1.5 - 2s
- **Watch out for:** neon-everywhere. The activation should reveal a *clear* UI underneath, not a wall of glow.

## 11. Split-Screen Feature Stack

Multiple screens slide in from different sides, stacking like cards (3-4 phones / dashboards in a fan or grid).

- **Best for:** multi-platform products (iOS+Android+web), feature-rich SaaS
- **Motion verbs:** *slide in from*, *stack with stagger*, *fan out*, *cascade*
- **Color cue:** consistent palette across all screens; one accent threads them together
- **Sound:** stagger of soft swipe sounds (0.15s apart) ending in a single chord
- **Duration:** 1.8 - 2.4s
- **Watch out for:** screens too small. Each must be readable; if you're showing 4+, drop to a focus-on-one-then-fan pattern instead.

## 12. Device from Void

A phone / laptop / tablet materializes from pure black, screen edges glowing as it solidifies.

- **Best for:** premium positioning, hardware launches, flagship product reveals
- **Motion verbs:** *materialize*, *fade in from black*, *solidify*, *glow at the edges*
- **Color cue:** pure black → brand palette on the screen; edge glow in primary accent
- **Sound:** subsonic build + satisfying "lock-in" chord on full reveal
- **Duration:** 1.5 - 2s
- **Watch out for:** too dark on small screens. Final brightness must be ≥60% — TikTok auto-skips dim openers.

## Chaining hooks (for 3-second openings)

If the spot is ≥20s, you can chain two hooks for a 3s cold-open. Combinations that work:

- **Code-to-Product → Particle Burst** (dev tool launch): code compiles, explodes into particles, particles form UI.
- **Glitch-to-Clean → UI Morph** (productivity / debugging): glitch resolves into a UI, then a UI element morphs to show the fix.
- **Data Explosion → Isometric Zoom** (BI / analytics ecosystem): metric explodes, particles re-gather as the platform map.
- **Device-from-Void → Hand-Interaction** (mobile app): phone materializes, hand reaches in and taps.

Don't chain three. By second 3, the showcase must already be running.

## Picking the right hook (decision shortcut)

| Product type | First-pick hook | Backup |
|---|---|---|
| SaaS dashboard / BI | Data-Viz Explosion | Isometric Camera Zoom |
| Mobile consumer app | Hand Interaction | Device from Void |
| AI / ML / data platform | Code-to-Product | Particle Burst from Logo |
| Dev tool / infra | Glitch-to-Clean | Code-to-Product |
| B2B enterprise platform | Isometric Camera Zoom | Particle Burst from Logo |
| Productivity / wellness | UI Morph | Hand Interaction |
| Gaming / crypto / music | Neon-Glow Activation | Light-Beam Scan |
| Security / cybersecurity | Light-Beam Scan | Glitch-to-Clean |
| Social proof / traction | Metric Counter Climb | Data-Viz Explosion |
| Multi-platform launch | Split-Screen Feature Stack | Device from Void |
