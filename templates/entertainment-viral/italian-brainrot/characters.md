# Italian Brainrot — character roster

33 canonical characters live in the `italian-brainrot-characters` pool of [ralphy-assets](https://github.com/alecs5am/ralphy-assets). Pull a canonical hero image for any of them with:

```bash
ralphy assets pull-pool italian-brainrot-characters/<slug> --install <project>
```

This is the **source of truth for the roster** — every slug here matches a pool entry exactly. To add a new character: add to the pool first (`ralphy-assets/pool/italian-brainrot-characters/<new-slug>.jpg` + manifest entry), THEN add a row here.

> Descriptors below come from each character's `brainrot.cyou` page (origin + trend context). See [ATTRIBUTION](https://github.com/alecs5am/ralphy-assets/blob/main/pool/italian-brainrot-characters/ATTRIBUTION.md) for license / take-down policy.

## Picking guide

By **scenario fit** — see [`prompt-cookbook.md` § Setting / context bank](prompt-cookbook.md#setting--context-bank).

By **call-sign already established** — see [`hooks.md`](hooks.md). Characters with established call-signs need no creativity; just lift the chant.

By **virality tier** — these had the most volume in the original trend wave: `tralalero-tralala`, `tung-tung-tung-sahur`, `ballerina-cappuccina`, `bombardiro-crocodilo`, `brr-brr-patapim`, `chimpanzini-bananini`. Safe picks for a first project.

## Adding a new character

If the user wants a character not in the 33: **add it to the pool first**, don't improvise.

```bash
# 1. Find a canonical source image (brainrot.cyou is the current aggregator)
# 2. Add to ralphy-assets repo:
cd ~/github/ralphy-assets
cp <source>.jpg pool/italian-brainrot-characters/<new-slug>.jpg
# 3. Update manifest.json with sha256 + sourceUrl + descriptor + worksWith=['italian-brainrot']
# 4. Commit + push
# 5. From ugc-cli: regenerate the catalog
ralphy assets catalog --write
```

Then add the row in this file. The canonical visual identity must exist BEFORE scenario writing — that's the AGENTS.md hard invariant #3 expressed in pool terms.
