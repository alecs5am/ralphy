# Voice mode: `sarcastic-aside`

**When:** Meme commentary, eye-roll punchline, deadpan-but-with-an-edge. Used for the second beat after a setup, NOT the hook.

## ElevenLabs settings

| Setting | Value |
|---|---|
| `stability` | 0.45 |
| `similarity_boost` | 0.80 |
| `style` | 0.30 |
| `use_speaker_boost` | true |

## Sample VO

> Oh sure, that'll definitely work this time. Because the last six attempts were just warm-ups, right? Anyway.

## Pacing target

~2.6 words/sec. Drag the sarcastic words (`sure`, `definitely`). Punch the close (`anyway`) flat.

## Do

- Slight pitch lift on the sarcastic word, then drop.
- Trail off on the closer — don't punch.

## Don't

- Smile-talk through it (kills the dryness).
- Pair with hype-music (kills the joke).
