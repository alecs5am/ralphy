# Voice mode: `intimate-reveal`

**When:** Confessional storytime, beauty / lifestyle reveal, "real talk" beats. Close-mic register, the listener feels two feet away.

## ElevenLabs settings

| Setting | Value |
|---|---|
| `stability` | 0.50 |
| `similarity_boost` | 0.90 |
| `style` | 0.40 |
| `use_speaker_boost` | false |

## Sample VO

> Okay so I wasn't going to talk about this but here we are. I tried it for two weeks before anything changed — and then it just clicked. I don't know what to tell you.

## Pacing target

~2.2 words/sec. Pauses uneven — natural thought-gathering. Audible swallow / lip-smack between sentences is desirable, not a defect.

## Do

- Lean toward the mic (or write the script as if leaning).
- One "okay so" / "alright" filler is permitted as a beat-setter.

## Don't

- Bright EQ. Roll off above 8kHz for the close-mic warmth.
- Music above -28 LUFS — should feel like silence behind the voice.
