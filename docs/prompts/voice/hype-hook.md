# Voice mode: `hype-hook`

**When:** First 0-3s scroll-stop hook. DTC ad, viral creator opener, "POV: you just…" delivery.

## ElevenLabs settings

| Setting | Value |
|---|---|
| `stability` | 0.30 |
| `similarity_boost` | 0.70 |
| `style` | 0.65 |
| `use_speaker_boost` | true |

## Sample VO

> Wait — you've been using this WRONG the whole time?! I tested fifteen of them and only one actually worked. Watch this.

## Pacing target

~3.0 words/sec on the punch words, then a beat. Front-load energy; the rest of the clip cools down.

## Do

- Rising intonation on the question hook.
- Audible breath intake before the punch word.
- Caps in script = louder + faster.

## Don't

- Sustained hype past 3s — fatigues fast.
- Whisper-loud-whisper pattern in a single hook (split into two).
