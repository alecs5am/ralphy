# Music mode: `deadpan-bg`

**When:** Bed under a deadpan-rant or talking-head-rant VO. Should be barely-there — listener notices only when it stops.

## Skeleton

```
<minimal lo-fi loop>, <60-90 BPM>, <one instrument: muted bass / soft kick>,
no melody, no fills, podcast-friendly, sits under VO
```

## Example prompts

1. Minimal lo-fi loop, 72 BPM, muted upright bass plays a 2-note ostinato, soft kick on the downbeat only, no melody, sits at the bottom of the mix.
2. Sparse jazz brush kit, 80 BPM, no rhythm section, occasional ride-cymbal swell, leaves the upper register empty for VO.
3. Ambient pad with one chord, no rhythm, very slow attack, 0 BPM (drone), sits under the VO without competing.
4. Soft analog synth bed, 68 BPM, one held bass note per bar, no top end, no melody.
5. Quiet vinyl-crackle texture, 70 BPM, gentle rim-shot every 4 bars, no melodic content.

## Don't

- Add a hook melody.
- Use stereo width — keep it center-anchored so it ducks cleanly.
- Exceed −24 LUFS in the final mix.
