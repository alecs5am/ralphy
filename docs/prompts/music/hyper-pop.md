# Music mode: `hyper-pop`

**When:** TikTok hyper-pop trend ride, ≤15s viral cut, scroll-stop energy.

## Skeleton

```
<hyper-pop loop>, <140-160 BPM>, <distorted 808s + bright synths + chopped vocal snippet (non-recognizable)>,
sidechain pump, short 4-bar loop, mixes loud
```

## Example prompts

1. Hyper-pop loop, 150 BPM, distorted 808 bassline, bright sawtooth synth lead, sidechain pump on the kick, no recognizable vocals, 4-bar loop.
2. Glitch-pop loop, 145 BPM, granular vocal chop (gibberish syllables only), 808 bass slides, bright pluck synth, hard sidechain.
3. Future-bass-tinged hyper-pop, 155 BPM, supersaw chord stabs, 808 sub, no vocal samples, 4-bar loop.
4. Rage-style hyper-pop, 160 BPM, distorted 808s, hard-clipped kick, no melody, percussive only, 2-bar loop.
5. Bubblegum-bass loop, 150 BPM, pitched-up vocal chop (non-lyrical), candy-bright synth lead, 808 bass, 4-bar loop.

## Don't

- Reference any artist or song.
- Use a recognizable hook (copyright + dilution risk).
- Mix quietly — this register only works hot.
