## Defaults

| Setting | Default | Notes |
|---|---|---|
| Model | `eleven_multilingual_v2` | EN + RU; switch to `eleven_v3` for shorter latency on EN |
| Stability | `0.55` | Lower = more variation (cinematic, emotional); higher = monotone (PSA, robo) |
| Similarity boost | `0.8` | Closer to the cloned voice; lower for more interpretation |
| Style | `0.25` | 0 = monotone broadcast, 1 = full dramatic |
| Speaker boost | `on` | Turn off for monotone broadcast / robo registers |

## Register presets

| Register | Stability | Similarity | Style | When to use |
|---|---|---|---|---|
| Cinematic / emotional | `0.45` | `0.85` | `0.6` | Brand story, transformation arcs, before-after testimonials |
| Default UGC selfie | `0.55` | `0.8` | `0.25` | Most product reviews, lifestyle content |
| Hype / energetic | `0.4` | `0.75` | `0.55` | Hormozi-style talking-head, hype reels |
| Deadpan / monologue | `0.7` | `0.85` | `0.1` | Deadpan POV, dry-humor commentary |
| Robo / PSA / analog horror | `0.85` | `0.9` | `0.0` (with `--no-speaker-boost`) | Analog-horror PSA, news-broadcast register |

## Runnable snippet

```bash
ralphy generate voiceover \
  --project espresso-001 \
  --slot scene-01-vo \
  --voice <your-elevenlabs-voice-id> \
  --text "Three mornings into the new machine and I haven't burned a single cup. Here's what changed." \
  --stability 0.55 \
  --similarity-boost 0.8 \
  --style 0.25
```

## Language notes

- **English** is the safe default. Both `eleven_multilingual_v2` and `eleven_v3` deliver consistent quality.
- **Russian** works on `eleven_multilingual_v2` but voice-age and accent can drift on long lines — keep each line ≤ 12 words. The Kling `--audio` flag in particular *does not* reliably handle RU (use ElevenLabs explicitly).
- Other languages: test with a short probe before committing to a project.

See [`voice exists`](/cli/overview) to verify a voice ID is callable on your ElevenLabs account before submitting.
