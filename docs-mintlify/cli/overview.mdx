`ralphy` is the main CLI. Once installed, it works from any folder — it locates the project via
`~/.config/ralphy/config.json` or by walking up from `cwd`.

```bash
ralphy --help
```

## Inside the project root

If you cloned the repo and have Bun installed, every command is also available via `bun run`:

```bash
bun run dev                   # Remotion Studio
bun run setup                 # = ralphy setup
bun run doctor                # = ralphy doctor
bun run ralphy -- <args>      # = ralphy <args>
bun run lint                  # eslint + tsc
```

## CI / non-interactive

The wizard needs a TTY. In CI, link the project explicitly and read the JSON status:

```bash
ralphy setup --link "$PWD" --status
```
