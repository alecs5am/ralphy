<Note>
  Ralphy is a CLI + Remotion composition library + AI-agent skill bundle, packaged as one binary. Two API keys, two commands, a chat with your agent — and an mp4 lands on disk.
</Note>

## Choose your path

The four most common ways people pick Ralphy up. Each path is 3 short reads.

<CardGroup cols={2}>

<Card title="🎬 I want to ship one video" icon="film" href="/quickstart">
The default path. Install → setup → first project → first render.

1. [Quickstart](/quickstart) — 5 steps, ~8 minutes
2. [Concepts](/concepts) — the 8 pieces of the mental model
3. [Cookbook: image prompts](/authoring/cookbook/image) — paste-edit-ship
</Card>

<Card title="🚂 I want to ship a batch" icon="layer-group" href="/cli/overview">
Producer mode — N variants from one template, parallel generation, rolled-up cost.

1. [Templates](/authoring/templates) — find a vibe template that fits
2. [`ralphy batch`](/cli/overview) — submit N projects atomically
3. [Producer playbook](https://github.com/alecs5am/ralphy/blob/main/docs/playbooks/producer.md) — the orchestration discipline
</Card>

<Card title="🤖 I want to plug in as an agent" icon="robot" href="/cli/overview">
Drive Ralphy from Claude Code / Cursor / Codex. Install the skill bundle once; the agent reads the playbooks and routes by user intent.

1. [`ralphy skill install`](/cli/overview) — claude / cursor / codex adapters
2. [`AGENTS.md`](https://github.com/alecs5am/ralphy/blob/main/AGENTS.md) — playbook routing table (the source of truth)
3. [Concepts](/concepts) — workspace, project, gen-log, manifest
</Card>

<Card title="✍️ I want to author templates" icon="pen-ruler" href="/authoring/templates">
Build your own vibe-style or vibe-reference template, register it, ship variants.

1. [Authoring templates](/authoring/templates) — `template.yaml` schema + composition.md
2. [Authoring skills](/authoring/skills) — slash-command extensions for agent flows
3. [`ralphy clone <url>`](/cli/overview) — mint a template from a real reference
</Card>

</CardGroup>

## What you'll need

<CardGroup cols={2}>
  <Card title="OpenRouter key" icon="key">
    One key for the media stack — image, video, LLM, vision, transcribe.
  </Card>
  <Card title="ElevenLabs key" icon="microphone">
    Voiceover (`eleven_multilingual_v2`) plus ElevenLabs Music for score.
  </Card>
  <Card title="Bun + ffmpeg" icon="terminal">
    Bun runs the CLI; ffmpeg is what Remotion shells out to.
  </Card>
  <Card title="~$10 per render" icon="coins">
    ~8 minutes per video, ~$8–12 in API costs on the default stack.
  </Card>
</CardGroup>
